# Multilingual News Summarization

This project implements a multilingual news summarization system using transformer-based models. It compares **mBART** and **mT5** for same-language summarization and provides a simple UI for interaction.

---

## Features

- Multilingual summarization (English, Spanish, French, Russian, Portuguese)
- Model comparison (mBART vs mT5)
- Optional translation using NLLB
- Simple React-based UI
- FastAPI backend for inference

---

## Tech Stack

- **Backend:** Python, FastAPI, Hugging Face Transformers  
- **Frontend:** React  
- **Models:** mBART-50, mT5-base, NLLB
